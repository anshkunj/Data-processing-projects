# Problem Statement

A company receives monthly Excel reports from multiple departments.
Each file has the same type of data but is stored separately.

Manually merging and summarizing these Excel files is time-consuming and error-prone.

### Challenges:
- Multiple Excel files in a folder
- Inconsistent column naming
- Need for automated summary generation

### Objectives:
- Read all Excel files from a directory
- Standardize column names
- Merge data into a single dataset
- Generate a summary report
- Export final Excel file with multiple sheets

### Expected Output:
- One consolidated Excel report
- Separate sheets for:
  - Raw merged data
  - Department-wise summary
